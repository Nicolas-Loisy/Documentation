# **Sage X3**

## **Fonctions**
- **Société** : GESCPY
- **Site** : GESFCY
- **Article** : GESITM
- **Site Article** : GESITF
- **Client** : GESBPC
- **Devis** : GESSQH
- **Commande** : GESPOH
- **Facture** : GESPIH

## **Annotations**
- **P** : Champ spécifique de type numérique ou texte.

## **Types de Champs**
- **A** : Alpha Numérique
- **ITM** : Article

## **Gestion des Tables**
- **Annulation** : Politique de suppression d'une table.
- **R.A.Z** (Remise à Zéro) :
  - Signification : **Rien À Zignaler**. (*Humour*)
  - Signification alternative : **Remise à Zéro**.

## **Création des Tables**
- La clé primaire doit être composée des 3 premières lettres suivies d'un 0.
- Toujours cliquer sur le bouton **VALIDATION** pour exécuter la requête `CREATE TABLE ...`.

## **Requêteur Graphique**
- Pour filtrer les données :
  - **Egal** : Utiliser la syntaxe `[F:XX] champfiltre=valeur`.
  - **Différent** : Utiliser la syntaxe `[F:XX] champfiltre<>valeur`.
- Pour ajouter des critères : Utiliser le bouton avec le logo **B**.


## Les droits utilisateurs

- Droits sur les fonctions (ex: création de commandes)
- Droits sur la visibilité dans la fonction
- Le filtre est fin

Dans Paramétrage -> Utilisateurs : Profil fonction

Dans Paramétrage -> Utilisateurs -> Habilitation fonctionnelle : Filtre des droits => Regroupement pour filtrer les sites
